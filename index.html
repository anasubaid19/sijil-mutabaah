Selamat Datang di DompetKu!

Aplikasi pencatatan keuangan pribadi offline yang aman, cepat, dan tanpa iklan. Data Anda 100% tersimpan di dalam perangkat Anda.

Petunjuk Pemakaian

Cara Install (iPhone / iOS)

  1. Buka aplikasi Safari.
  2. Ketuk ikon Share (Bagikan) di tengah bawah layar.
  3. Pilih "Add to Home Screen" (Tambah ke Layar Utama).
  4. Ketuk "Add" di pojok kanan atas.

Cara Install (Android)

  1. Buka browser Chrome.
  2. Ketuk ikon Tiga Titik di pojok kanan atas.
  3. Pilih "Tambahkan ke Layar Utama" atau "Install App". Jika Android masih membaca sebagai browser biasa, pastikan Anda memuat aplikasi ini dari server yang memiliki HTTPS.

Backup & Restore Data

  • Data Tersimpan di HP: Aplikasi ini offline. Data Anda aman tapi bisa hilang jika Anda menghapus cache browser secara ekstrem.
  • Backup: Pergi ke Pengaturan > Export Data > File JSON. Simpan file ini baik-baik.
  • Restore: Saat ganti HP, buka Pengaturan > Import Data > Pilih file JSON tadi.
